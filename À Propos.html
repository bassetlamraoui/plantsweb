À Propos de PhytoLex

L'Encyclopédie Des Plantes

PhytoLex est une base de données en ligne de specimens de plantes numérisés. C'est une grande liste des specimens de plantes connues.

Notre application est aussi équipée d'un puissant moteur de recherche avec lequel vous pouvez trouver n'importe quelle plante dans notre base de données en ecrivant le nom commun, le nom botanique, l'espèce, la famille...

PhytoLex fournit des informations précises, standardisées et vérifiées sur les plantes avec leurs photos et une description détaillée. Les informations incorrectes peuvent êtres signalées par l'utilisateur dans la section 'Feedback'.

Les utilisateurs peuvent également parcourir notre vaste collection de plantes dans la section "Découvrir".
